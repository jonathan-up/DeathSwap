# DeathSwap
A mini game for spigot server

Just like [Dream](https://www.youtube.com/watch?v=vXS1pXWslxs)

### Tested:
* 1.16
* 1.17
* 1.18
* 1.19

## Usage
* Download jar file at release
* Put it to your plugins folder
* And run your server

## For developer
### build
* `git clone https://github.com/jonathan-up/DeathSwap.git`
* `mvn package`
